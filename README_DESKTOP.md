## ? Troubleshooting

### Browser doesn't open?
Manually go to: https://localhost:7223/index.html

### SSL warning in browser?
Click: "Advanced" ? "Proceed to localhost (unsafe)"

### Port already in use?
Close any running AdminMembers server windows and try again

### .NET not found?
Install from: https://dotnet.microsoft.com/download/dotnet/8.0
