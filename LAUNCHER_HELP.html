?? How to Use

  1. Right-click on ?? START HERE.bat
  2. Select "Send to" ? "Desktop (create shortcut)"
  3. Double-click the shortcut on your desktop
  4. Wait a few seconds for the server to start
  5. Browser opens automatically - Start using the app!

?? How to Stop

To stop the AdminMembers server:

  • Close the "AdminMembers Server" window
  • Or press Ctrl+C in that window
