## ?? MEEST VOORKOMENDE ERRORS

### Error: "Failed to fetch"
```
Betekenis: Backend is niet gestart
Fix:      Visual Studio ? F5
Test:     Refresh autotest.html ? Run tests
```

### Error: "HTTP 500"
```
Betekenis: Database probleem
Fix:      Package Manager Console ? Update-Database
Test:     Run tests opnieuw
```

### Error: "Backup FAIL"
```
Betekenis: Backup endpoint probleem
Fix:      Check Visual Studio Output voor details
          Mogelijk: Database permissions
Test:     Run tests opnieuw
```
